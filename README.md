# TapTalk

TapTalk is an AAC iOS app built with Expo Router and React Native. It targets iOS 16+ and is designed to run in Expo Go during development.

## Run On Your Phone

1. Install dependencies with `npm install`.
2. Start Expo with `npx expo start`.
3. Open Expo Go on your iPhone and scan the QR code from the terminal.

The current auth and subscription screens are local mock flows. Supabase is intentionally not connected yet.
