Clo — the baby-blue speech-bubble mascot. Use for onboarding moments, encouragement, empty states, and emotional feedback. Path-agnostic: pass the resolved `src`.

```jsx
<Mascot src="assets/mascots/happy_smile.png" alt="Clo smiling" size={140} float />
```

~30 poses ship under `assets/mascots/` (happy_smile, excited_wave, thinking_puzzled, sad_worried, sleeping, heart_eyes, winking_smile, neutral_curious, …). Always set a meaningful `alt`. `float` adds a calm idle bob.
