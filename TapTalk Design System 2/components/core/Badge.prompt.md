Compact status badge / tag for streaks, counts, and plan labels.

```jsx
<Badge tone="success">Free forever</Badge>
<Badge tone="primary" solid>PREMIUM</Badge>
<Badge tone="warning">3 day streak</Badge>
```

Tones: `neutral` / `primary` / `success` / `warning` / `danger`. `solid` gives a filled white-text variant; default is a soft tint.
