White rounded content surface with a subtle shadow — the default container for grouped content.

```jsx
<Card>…rows…</Card>
<Card radius="bg" padding={24}>…large background card…</Card>
<Card radius="sheet" border={false}>…bottom sheet…</Card>
```

`radius`: `card` (16px) / `bg` (40px) / `sheet` (44px). Toggle the hairline `border` and set `padding`.
